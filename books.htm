IBM Technical Reference Manuals (part of the IBM Personal Computer Hardware Reference Library):

  • IBM PC Technical Reference (Aug 1981)
  • IBM PC Technical Reference (Apr 1984)
  • IBM PC XT Technical Reference (Apr 1983)
  • IBM PC XT and Portable PC Technical Reference (Mar 1986)
  • IBM PC AT Technical Reference (Mar 1984)
  • IBM PC AT Technical Reference (Sep 1985) - released with AT model 239
  • IBM PC AT Technical Reference (Mar 1986) - released with AT models 319 and 339
  • IBM PC XT 286 Technical Reference (Aug 1986)

PC BIOS books:

  • IBM PS/2 and PC BIOS Interface Technical Reference (Apr 1987, Sep 1991)
  • System BIOS for IBM PC/XT/AT Computers and Compatibles by Phoenix Technologies (first edition - 1989)
  • System BIOS for IBM PCs, Compatibles, and EISA Computers by Phoenix Technologies (second edition - 1991)
  • Programmer's Guide to the AMIBIOS by American Megatrends (1993)

DOS books:

  • The Peter Norton Programmer's Guide to the IBM PC by Peter Norton (first edition - 1985)
  • Advanced MS-DOS Programming by Ray Duncan (1986)
  • The New Peter Norton Programmer's Guide to the IBM PC & PS/2 by Peter Norton & Richard Wilton (second edition - 1988)
  • The MS-DOS Encyclopedia (1988) - covers up to DOS 3.3
  • The Programmer's PC Sourcebook by Thom Hogan (first edition - 1988) - covers up to DOS 3.3
  • The Programmer's PC Sourcebook by Thom Hogan (second edition - 1991) - covers up to DOS 5.0
  • Microsoft MS-DOS Programmer's Reference (DOS 5) (1991)
  • Microsoft MS-DOS Programmer's Reference (DOS 6) (1993)

Other books:

  • The C Programming Language, Second Edition by Brian Kernighan & Dennis Ritchie (1988)

Intel 80x86 reference manuals:

  • Intel 8086 Family User's Manual
  • Intel iAPX 286 Programmer's Reference Manual
  • Intel 386 Programmer's Reference Manual
  • Intel 486 Processor Programmer's Reference Manual
  • Intel Pentium Processor Family Developer's Manual (3 volumes)
  • Intel Pentium Pro Family Developer's Manual (3 volumes)
  • Intel IA-32 Intel Architecture Software Developer's Manual (4 volumes) - last printed version of the Intel documentation in 2004

Current x86 documentation links:

  • Intel - https://www.intel.com/content/www/us/en/developer/articles/technical/intel-sdm.html

Also, Microsoft published several versions of the Microsoft Programmer's Library CD-ROM which included machine-readable versions of a number of books. The last known release (1.4) was from July 1991 (List of contents)

Finally, while not official, Ralf Brown's Interrupt List was a comprehensive listing of PC interrupt calls, I/O ports, memory locations, far-call interfaces, both documented and undocumented.
